bcRead: Automatic barcode reading¶

class libs.bcRead.bcRead(patterns, rotation_list=[9, 25, 18], parent=None, *args)¶

A barcode reader class

Args: prefix (str, optional): a collection code prefix. If left empty,

Matches any to no value(s)

digits (int, optional): the quantity of digits expected in the barcode.

If left empty, matches any to no digit(s)

rotation_list (iterable, optional): iterable containing a series of int

representing image rotations (in degrees) to attempt if no barcode is found. Default values are [9, 15, 18]. Rotation attempts stop after any results are found. The list’s rotations are cumulative. Short or empty lists will decrease the time before giving up on finding a barcode.

Attributes: rePattern (obj): A compiled regex pattern rotation_list (list): The saved rotation list
